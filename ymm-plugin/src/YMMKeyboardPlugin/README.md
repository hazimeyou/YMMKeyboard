# YMMKeyboardPlugin

YMMKeyboardPlugin は、YMMKeyboard リポジトリ内の YukkuriMovieMaker 4 向けプラグイン本体です。
YMMKeyboard本体の入力と、YMM 内の UI キーボード入力をタイムラインに反映させます。

親ドキュメント: [`../README.md`](../README.md)

## 主な機能

- YMMKeyboard本体やUIからの入力受信とキー割り当て
- 全アイテム（テンプレート）のタイムライン配置
- 自由なレイヤー配置
- タイムラインシーク
- 実機、UIごとのキー割り当て
- 単独キーと複数キー組み合わせの割り当て

## 起動方法
リリースから YMMKeyboardPlugin.ymme をダウンロードし、実行してください。（現在未リリース）
ツールタブに「キーボードプラグイン」というボタンが追加されます。（名称変更予定）

## キー割り当て
キー割り当ては、YMMの設定画面→キーボードプラグイン内の割り当てを開くから行います。

## 設定保存先

設定はプラグイン DLL と同じフォルダー配下に保存されます。

- フォルダー: `settings`
- ファイル: `YMMKeyboardSettings.json`
