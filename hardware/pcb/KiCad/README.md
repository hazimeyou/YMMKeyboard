# KiCad

キーボード本体のハードウェアCADファイルです。

## 構成

- `YMMKeyboard/` : 初期型キーボードのKiCadプロジェクトファイルと製造データ
- `YMMKeyboardMX/` : CherryMX互換キー搭載型キーボードのKiCadプロジェクトファイルと製造データ

## メモ

`YMMKeyboard`は安価なキーを使用した初期型キーボードのプロジェクトです。jlcpcbで注文することを前提に100x100mmの基板サイズに収まるように設計しています。

`YMMKeyboardMX`は実際に公開当を想定したCherryMX互換キーを使用したキーボードのプロジェクトです。こちらは使いやすさのため100x100mmを超えてます。
