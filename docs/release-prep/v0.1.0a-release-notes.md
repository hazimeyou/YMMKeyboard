# v0.1.0a Release Notes Draft

## Overview

`YMMKeyboard` is a keyboard input system for `YukkuriMovieMaker 4 (YMM4)` built around `RP2040`, `USB HID`, matrix scan input, and rotary input support.

This draft freezes the `v0.1.0a` scope for the first public release candidate. It does not change runtime behavior and is intended only as release preparation material.

## Main Features

- RP2040 support
- USB HID support
- Matrix keyboard support
- Rotary encoder support
- YMM4 operation support
- Legacy COM diagnostic fallback

## Supported Environment

- Windows host for the YMM plugin
- YukkuriMovieMaker 4 environment
- RP2040-based firmware target
- USB connection for HID input

## Known Limitations

- COM / CDC / Serial are not the primary input path
- Legacy COM diagnostic fallback is kept for observation and compatibility only
- This release candidate does not add new macros, new devices, or advanced settings UI
- Auto-update is not included

## Installation

1. Install or prepare the YMM4 environment.
2. Place the YMM plugin in the expected plugin location.
3. Flash the firmware UF2 to the RP2040 device.
4. Connect the device over USB and confirm HID input is recognized.
