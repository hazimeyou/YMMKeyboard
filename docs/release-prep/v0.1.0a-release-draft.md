# YMMKeyboard v0.1.0a Release Draft

## Overview

YMMKeyboard is a keyboard input system for YukkuriMovieMaker 4 (YMM4). The `v0.1.0a` release is the first public release candidate and focuses on the established input path rather than new functionality.

## Main Features

- RP2040 support
- USB HID support
- Matrix keyboard support
- Rotary encoder support
- YMM4 operation support
- Legacy COM diagnostic fallback

## Supported Environment

- Windows 10
- Windows 11
- YukkuriMovieMaker 4
- RP2040-based device
- USB connection for HID communication

## Distribution Artifacts

- `YMMKeyboardPlugin.dll`
- `ymm_keyboard_fw.uf2`
- `README.md`
- `LICENSE`

Suggested package name:

- `YMMKeyboard-v0.1.0a.zip`

## Installation

1. Copy or install `YMMKeyboardPlugin.dll` into the YMM4 plugin location used by the current setup.
2. Flash `ymm_keyboard_fw.uf2` to the RP2040 device.
3. Start YMM4 and confirm the plugin is recognized.
4. Verify key input and rotary input on the connected device.

## Known Limitations

- RP2040 firmware is required for the primary input path.
- Legacy COM diagnostic fallback is kept for observation and compatibility only.
- This release does not introduce macro expansion, additional devices, advanced settings UI, or auto-update.
