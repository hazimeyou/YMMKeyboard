# YMMKeyboardPlugin

YMMKeyboardPlugin は、YMMKeyboard リポジトリ内の YukkuriMovieMaker 4 向けプラグイン本体です。
YEKBの入力と、YMM 内の UI キーボード入力を、ユーザーが自由にキー割り当てできるようにすることを目的としています。

親ドキュメント: [`../README.md`](../README.md)

## 主な機能

- YEKBやUIからの入力受信とキー割り当て
- 音声アイテムのタイムライン配置
- タイムラインシーク
- COM ポートの選択、接続、接続解除
- 起動時に接続する COM ポートの複数登録
- 実機 UID ごとのキー割り当て
- UI キーボード用の独立したキー割り当て
- 単独キーと複数キー組み合わせの割り当て

## 起動方法
リリースから YMMKeyboardPlugin.ymme をダウンロードし、実行してください。
ツールタブに「キーボードプラグイン」というボタンが追加されます。（名称変更予定）

## キー割り当て
キー割り当ては、YMMの設定画面→キーボードプラグイン内の割り当てを開くから行います。

## 設定保存先

設定はプラグイン DLL と同じフォルダー配下に保存されます。

- フォルダー: `settings`
- ファイル: `YMMKeyboardSettings.json`

## メモ

- 実機の複数キー判定は、単独キーを少し待ってから確定する方式です。
- UI キーボードと実機は設定が別です。
- アクション追加時は [`Mapping/MappingConverter.cs`](/Mapping/MappingConverter.cs) を入口に増やしていく想定です。
- YEKBは仮の名称です。正式名称は未定です。
- 今後の予定
- すべてのアイテム操作とプラグイン間連携を実装予定です。
- YEKB用実行スクリプト（プラグイン）を追加予定です。
